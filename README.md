# Illustrator Scripts

Coleção de scripts JSX para Adobe Illustrator, criada para automatizar tarefas repetitivas no fluxo de trabalho de design.

## Scripts disponíveis

### Renomear artboards a partir de uma lista

Permite colar uma lista de nomes e aplicar cada linha a um artboard, seguindo a ordem do documento. É uma forma simples de padronizar nomes antes da exportação, especialmente em arquivos com muitas pranchetas.

### Renomear artboards com tamanho

Adiciona automaticamente as dimensões do artboard ao final do nome. Isso é útil para exportação de peças em formatos variados, porque facilita a identificação de cada prancheta já no nome do arquivo.

## Como usar

Os arquivos `.jsx` podem ser executados diretamente pelo Illustrator em:

`File > Scripts > Other Script...`

Também é possível instalar os scripts na pasta de Scripts do Illustrator para que apareçam diretamente no menu do programa.

## Sugestão de instalação no Windows

`C:\Program Files\Adobe\Adobe Illustrator [versão]\Presets\pt_BR\Scripts`

## Boas práticas

- Fechar e abrir o Illustrator depois de copiar os scripts para a pasta.
- Manter nomes simples, sem caracteres especiais em excesso.
- Revisar os nomes antes de exportar.

## Observação

Os scripts foram criados para uso pessoal e profissional e podem ser adaptados conforme o fluxo de trabalho de cada designer.
